# FUTURE_DS_02
# Customer Churn Analysis Dashboard

## 📊 Project Overview

This project focuses on analyzing telecom customer data to understand **customer churn behavior** and identify key factors that influence customer retention.

The objective of this project is to transform raw customer data into **actionable insights using Excel and Power BI**. By analyzing customer demographics, contract types, services, and payment methods, businesses can better understand why customers leave and take steps to improve retention.

---

## 🎯 Project Goals

* Analyze customer churn patterns in a telecom dataset
* Identify key drivers that lead to customer churn
* Build an interactive **Power BI dashboard** for business insights
* Provide data-driven recommendations to improve customer retention

---

## 📂 Dataset

The dataset used in this project is the **Telco Customer Churn Dataset**.

It contains customer information such as:

* Customer ID
* Gender
* Senior Citizen
* Contract Type
* Internet Service
* Payment Method
* Tenure
* Monthly Charges
* Total Charges
* Churn Status

This dataset helps analyze **customer behavior and subscription patterns**.

---

## 🧹 Data Cleaning (Excel)

Before building the dashboard, the dataset was cleaned and prepared using **Microsoft Excel**.

Key steps included:

* Handling missing values in the **TotalCharges** column
* Converting data types from text to numeric
* Creating additional columns such as **Tenure Group**
* Validating and preparing the dataset for Power BI analysis

---

## 📈 Power BI Dashboard

An interactive **Customer Churn Analysis Dashboard** was built using Power BI.

### Key KPIs

* Total Customers
* Churn Customers
* Churn Rate
* Total Revenue
* Average Monthly Charges

### Visualizations

The dashboard includes multiple visuals to analyze churn patterns:

* **Churn Customers by Contract Type**
* **Customer Distribution by Tenure Group**
* **Customer Distribution by Internet Service**
* **Churn Customers by Payment Method**
* **Average Monthly Charges by Churn Status**

Interactive **filters (slicers)** allow users to explore data by:

* Contract Type
* Internet Service
* Gender

---

## 🔍 Key Insights

From the analysis, several important insights were identified:

* Customers with **Month-to-Month contracts show the highest churn rate**
* Customers with **higher monthly charges tend to churn more frequently**
* Customers with **longer tenure are more loyal and less likely to churn**
* **Fiber optic internet users show higher churn compared to DSL users**
* Certain **payment methods are associated with higher churn rates**

These insights can help businesses design better **customer retention strategies**.
